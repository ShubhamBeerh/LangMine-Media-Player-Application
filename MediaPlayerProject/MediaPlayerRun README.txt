*********************************************INSTRUCTIONS FOR RUNNING THE MEDIA PLAYER APPLICATION*******************************************
1)1)Install JDK 8   :-> https://www.oracle.com/in/java/technologies/javase/javase-jdk8-downloads.html
2)Open Environment Variables and go to System Variables.
3)Set Variable Name = JAVA_TOOL_OPTIONS and Variable Value = -Dfile.encoding=UTF8.
4)Double Click on LangMine JAR present in Application Folder.
